# Normativas peruanas

Esta carpeta concentra la base legal y tecnica que sustentara el mini proyecto.

## Orden sugerido

1. **Fuentes oficiales:** enlaces, entidad emisora, fecha y estado de vigencia.
2. **Fichas de norma:** resumen corto de lo que aplica a una vivienda.
3. **Matriz de cumplimiento:** relacion entre requisito normativo, decision del proyecto y evidencia.
4. **Anexos:** capturas, extractos permitidos, tablas propias y referencias bibliograficas.

## Criterio de uso

- Priorizar paginas oficiales del Estado peruano: `gob.pe`, MINEM, MVCS, SENCICO u OSINERGMIN.
- No copiar normas completas dentro del repositorio si no es necesario; guardar enlace oficial y resumen propio.
- Citar articulo, numeral, capitulo o seccion cuando se use un requisito para justificar una decision.
- Diferenciar entre requisito obligatorio, criterio recomendado y dato pendiente de verificar.
