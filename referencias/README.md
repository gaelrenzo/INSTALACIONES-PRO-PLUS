# Referencias

## Normativa versionada

- `normativa/pdf/`: documentos normativos en PDF.
- `normativa/documentacion/`: fichas, notas y extractos de apoyo.

## Material local

`local/` contiene colecciones pesadas, fotografías, proyectos guía y documentación de consulta disponible en esta estación de trabajo. Está ignorado por Git y no debe asumirse presente en otros clones.

Las referencias sirven como evidencia y consulta. No deben modificarse para adecuarlas al resultado de un cálculo ni mezclarse con las entradas canónicas de un proyecto.
