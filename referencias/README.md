# Normativas y Material de Referencia (referencias/)

Esta carpeta contiene la documentación regulatoria, tablas técnicas oficiales y catálogos de fabricantes que fundamentan los cálculos y simbologías del sistema.

## Contenido

*   **Normativa Nacional:** Código Nacional de Electricidad (CNE - Utilización), Reglamento Nacional de Edificaciones (RNE - Norma EM.010).
*   **Catálogos:** Fichas técnicas de conductores, interruptores termomagnéticos y diferenciales de proveedores locales.
*   **`local/`**: Carpeta destinada a archivos PDF pesados, manuales completos o bases de datos grandes. **Nota:** El contenido de esta carpeta está ignorado en Git para mantener ligero el repositorio.

## Regla de Uso
Cualquier cálculo o símbolo utilizado por las herramientas debe estar referenciado y justificado bajo la documentación contenida o citada en este directorio.
