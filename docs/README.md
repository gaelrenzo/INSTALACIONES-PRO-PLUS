# Documentación del Sistema (docs/)

Esta carpeta contiene la documentación técnica y metodológica sobre el diseño, arquitectura y flujo de agentes de IA en el proyecto.

## Contenido

*   **[arquitectura.md](file:///docs/arquitectura.md):** Describe los principios del sistema, la separación estricta entre fuentes, datos canónicos y resultados, y el esquema de directorios por proyecto.
*   **[flujo-agentes.md](file:///docs/flujo-agentes.md):** Detalla los pasos del ciclo de vida del diseño (ingesta, extracción, validación geométrica, cálculo de cargas, planos CAD, cotización y control de calidad humano).
*   **Políticas de Datos:** Reglas de almacenamiento de datos canónicos y respaldos.

## Pautas
Antes de realizar cualquier modificación al código central o al pipeline de automatización, los agentes de IA y desarrolladores deben revisar la arquitectura descrita aquí para mantener la consistencia del sistema.
