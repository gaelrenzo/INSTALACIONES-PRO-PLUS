# Ejemplos CAD

`layout.json` — ejemplo de layout arquitectónico compatible con `dxf_generator.py`.

Contiene:
- `dimensions`: dimensiones totales del plano
- `rooms`: lista de ambientes con nombre, posición (x, y), ancho, alto
- `walls`: muros manuales adicionales
- `doors`: puertas con orientación y giro
- `windows`: ventanas
- `stairs`: escaleras
- `fixtures`: mobiliario (camas, mesas, cocina, baño)
- `custom_dimensions`: cotas personalizadas
