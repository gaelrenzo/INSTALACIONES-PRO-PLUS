# Plantillas de Configuración y CAD (plantillas/)

Directorio para almacenar los archivos base estandarizados que se utilizan para iniciar nuevos proyectos o estructurar reportes.

## Contenido

*   **`proyecto_template.yaml`**: Plantilla para el archivo de configuración inicial de cualquier nuevo proyecto.
*   **Plantillas CAD (.dxf)**: Dibujos con cajetines, marcos y rótulos en diferentes formatos (A3, A2, A1) configurados con capas técnicas estándar.
*   **Plantillas de Reportes (.md / .docx)**: Formato de estructura para memorias descriptivas y de cálculo.
