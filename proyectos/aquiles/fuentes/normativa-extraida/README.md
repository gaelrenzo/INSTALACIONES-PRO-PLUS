## Criterio de revision

- Si el PDF tiene texto extraible, se puede usar `pdftotext` para ubicar terminos y numerales.
- Si el PDF esta escaneado o cifrado contra copia, se extrae la pagina como imagen con `pdftoppm` y se revisa visualmente.
- Toda cita normativa usada en el informe debe quedar relacionada con una pagina extraida o con una ficha normativa del repositorio.
