# Expediente tecnico — Proyecto Aquiles

Fuentes editables del documento tecnico del proyecto.

| Subcarpeta | Contenido |
|------------|-----------|
| `capitulos/` | Capitulos del documento tecnico |
| `docs/` | Auditorias, correcciones, plan de correcciones, recalculos |
| `figuras/` | Imagenes y diagramas |
| `planos/` | Planos incluidos en el expediente |
| `tablas/` | Tablas incluidas en el expediente |

## Estructura del documento

1. Memoria descriptiva
2. Calculos justificativos
3. Especificaciones tecnicas
4. Metrados
5. Presupuesto
6. Planos
