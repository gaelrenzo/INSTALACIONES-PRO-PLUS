# Tests — Proyecto Renzo

Pruebas especificas del proyecto Renzo.

| Archivo | Prueba |
|---------|--------|
| `test_circuitos.py` | Validacion de circuitos electricos |
| `test_conductores.py` | Seleccion y verificacion de conductores |
| `test_demanda.py` | Calculo de maxima demanda |
| `test_presupuesto.py` | Calculo de presupuesto |

Ejecutar:

```bash
python3 -m pytest proyectos/renzo/tests
```
