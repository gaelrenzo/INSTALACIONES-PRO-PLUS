# Scripts del proyecto

- `construir_expediente.py`: orquestador principal.
- `generar_arquitectura.py`: convierte los tres layouts canonicos a DXF.
- `generar_planos_desde_modelo.py`: genera IE-02, IE-03 e IE-04.
- `generar_planos_y_diagramas.py`: genera IE-05 e IE-06 para el pipeline.
- `actualizar_metrados_latex.py`: actualiza capitulos de metrado y presupuesto.

Los scripts antiguos de creacion y migracion estan en
`../archivo/scripts-migracion/` y no forman parte del pipeline.
