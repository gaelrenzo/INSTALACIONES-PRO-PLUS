# Diseno electrico

La entrada activa es `datos/modelo-electrico.json`. El archivo relaciona cada
piso con su layout arquitectonico, circuitos, puntos electricos,
canalizaciones, tableros y puesta a tierra.

`diagramas/` contiene fuentes editables manuales. Los DXF y PDF generados van
a `build/renzo/`; solo las versiones aprobadas se copian a `../entregables/`.
