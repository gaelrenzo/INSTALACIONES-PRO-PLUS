# Modelos de datos

Modelos Pydantic que definen las estructuras canónicas del sistema.

| Archivo | Contenido |
|---------|-----------|
| `topologia.py` | `RedElectrica`, `Tablero`, `Circuito`, `Carga`, `Conductor` — representación completa de la red eléctrica |

Los modelos validan los datos de entrada y son el contrato entre dominios.
